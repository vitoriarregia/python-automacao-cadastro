### 📝 Sobre o Projeto
Como **Analista de Dados**, desenvolvi este projeto para unir a eficiência da automação (RPA) com a inteligência do processamento de dados. O objetivo principal é a extração de informações de bases externas e o input automatizado em sistemas de gestão, garantindo a integridade dos dados.

### 📊 Análise e Manipulação de Dados
Neste repositório, o foco analítico está na organização da base de dados:
* **Python & Pandas**: Utilizados para a leitura, limpeza e estruturação do arquivo `produtos.csv` antes da execução do bot.
* **Tratamento de Dados**: Lógica implementada para percorrer tabelas e garantir que cada campo (Código, Marca, Tipo) seja processado corretamente.
* **SQL (Interface)**: Estrutura preparada para futura integração com bases de dados relacionais, permitindo consultas e queries diretas para automação em larga escala.

---

### 🛠️ Stack Tecnológica
* **Python**: Core do projeto para lógica e automação.
* **SQL**: Modelagem lógica para persistência e consulta de dados.
* **Pandas**: Manipulação de DataFrames e análise de ficheiros CSV.
* **PyAutoGUI**: Automação de interface para input de dados processados.
