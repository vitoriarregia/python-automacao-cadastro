### 📝 Sobre este Projeto
Este é o meu **primeiro projeto prático de estudos** em Python. Como estou em transição para a área de tecnologia, desenvolvi este bot para aplicar conceitos de lógica de programação e manipulação de arquivos. O objetivo foi automatizar o cadastro de produtos a partir de uma base CSV, simulando uma tarefa real de rotina administrativa.

### ⚠️ Notas de Estudo (Configuração)
* **Aprendizado sobre Coordenadas:** Neste projeto, aprendi a usar o **PyAutoGUI** com cliques em eixos `x` e `y`.
* **Ajuste Técnico:** O código funciona com base na resolução do meu monitor. Para testar em outro computador, os pontos de clique precisam ser atualizados no script `Codigo.py`.

---

### 🚀 O que pratiquei neste projeto?
1. **Lógica com Python:** Estruturação de loops para percorrer listas de dados.
2. **Pandas para Iniciantes:** Como ler arquivos externos (.csv) e organizar informações em tabelas.
3. **Automação de Interface:** Controle de mouse, teclado e navegação web automática.
